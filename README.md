# Encore

Concert ticket platform.

## Prerequisites

- [Go 1.26+](https://go.dev/dl/)
- [golangci-lint v2](https://golangci-lint.run/welcome/install/)

## Stack

- **Programming Language:** Go 1.26
- **HTTP:** Chi


## Getting Started

```bash
# Clone
git clone https://github.com/TheTopPal/encore
cd encore

# Run
make run
```
